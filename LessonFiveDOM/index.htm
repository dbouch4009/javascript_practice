Section 4: JavaScript In The DOM

  • DOM: Document Object Model
  • DOM connects webpages to JS
  • For each HTML Box, there is a DOM object
  • JavaScript interacts/manipulates the DOM, but is not the DOM itself
  • HTML Document is what holds the content of a webpage. Elements are contained inside of opening and closing tags.
  • 'div' is the standard divider for layouts
  • Attributes like class, id, alt, dictate the styles and names
  • id attribute must be unique, but class names can be reused
  • Header contains page info, style info, etc., while Body contains page content
  • CSS contains rules to standardize style and layout of a page
  • * is universal selector, and can apply those styles to all classes in the style page
  • 'body' selector can apply styles to all elements in the body
